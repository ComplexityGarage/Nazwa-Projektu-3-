# Description of the project
This project is an arduino based walking model showcaseing a waling cycle of ants and other insects. Its equiped with 6 independent legs that can be controled via a bluetooth, as well as internar power source in form of batteries, allowing the model to funcion wirelessly.
# Science and tech used
- Arduino UNO R3
- HC-05 arduino bluetooth module
- Multiple small servomechnisms
- AA batteries and 9-volt battery as a power source
- [3D printed parts](placeholder)

# State of the art
This model is an original design inspired by other commonly used designs of limbs with 2 axis of rotation.  
![Photo of the model]()  
The movment itself (as mentioned many times before) is inspired by the walk cycle of an ant showedcased in: (https://www.youtube.com/watch?v=kNeMp8soxhI)

# What next?
This model can be used as an educational prop to showcase in a simplified yet interesting way the walk cycles of ants. If that ware the case its possible to make a decorative caseing in a shape of an ant or other insect for this model.  
Anothe future use for this model may be as a baseline for 6-legged walking robots, as this waling cycle can be used in difficult terrains thanks to its rather high stability.
